🎬 Hoe gebruik je de post-bestanden?

  1. Klik op "Open fullscreen" bij een post hieronder
  2. De post opent op exacte 1080×1920px (TikTok-formaat)
  3. Maak een screenshot met Cmd+Shift+4 of gebruik browser print → Sla op als PDF
  4. Open CapCut → Nieuw project → Voeg je kattenvideo toe als achtergrond
  5. Importeer de screenshot als overlay → pas opacity aan naar ~90% → export
  6. Upload direct vanuit CapCut naar TikTok onder account @VoerWijs
